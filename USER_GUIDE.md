# 🚀 User Guide - UAE Expat Finance Bot

Telegram: @expatsfinbot\
Main File: uae_expat_finance_bot.py\
Storage: Local JSON (user_profiles folder)

------------------------------------------------------------------------

# 📌 Overview

The UAE Expat Finance Bot helps you:

-   Track monthly income
-   Log expenses
-   Record savings
-   Maintain emergency fund data
-   View financial summaries
-   Monitor financial health

This version does NOT include: - AI financial advice - End of Service
(EOS) calculator - Purchase affordability engine - External API
integrations

------------------------------------------------------------------------

# 🟢 Getting Started

## Step 1 --- Start the Bot

1.  Open Telegram
2.  Search for: @expatsfinbot
3.  Tap Start or send `/start`
4.  Main menu will appear

A profile is automatically created for you.

------------------------------------------------------------------------

# 💰 Setting Up Your Financial Profile

## 1️⃣ Update Income

-   Tap **Update Income**
-   Enter your total monthly salary
-   Example: `15000`

✔ Use total take-home pay (including allowances)

------------------------------------------------------------------------

## 2️⃣ Add Monthly Expenses

Tap **Monthly Expenses** and enter values for:

### Essential Categories:

-   Rent
-   Food
-   Transport
-   Telecom
-   Utilities

### Lifestyle Categories:

-   Entertainment
-   Other expenses

✔ Enter only numbers\
✔ Use monthly averages if expenses vary

------------------------------------------------------------------------

## 3️⃣ Add Savings

-   Tap **Add Savings**
-   Enter how much you saved this month

Recommended target: - Minimum: 10% - Ideal: 15--20% of income

------------------------------------------------------------------------

## 4️⃣ Update Emergency Fund

-   Tap **Emergency Fund**
-   Enter how much you currently have saved

Recommended target: - 3 months of expenses (minimum) - 6 months (ideal)

------------------------------------------------------------------------

# 📊 Viewing Your Financial Status

Tap **Current Financial Status** to see:

-   Monthly income
-   Total expenses
-   Total savings
-   Remaining balance
-   Financial health indicators

Manual Formula Used:

Remaining Balance = Income - Total Expenses - Savings

------------------------------------------------------------------------

# 🎯 Budgeting Guidelines (UAE Context)

### 🏠 Rent (25--35% of income)

-   Keep under 40% maximum

### 🍽 Food (15--20%)

-   Track groceries + dining

### 🚗 Transport (5--10%)

-   Metro is cheaper than taxis
-   Car ownership increases monthly burden

### 📱 Telecom (2--3%)

-   Review packages annually

### ⚡ Utilities (5--8%)

-   Higher in summer due to AC

### 💰 Savings (10--20%)

-   Increase savings rate with each salary raise

------------------------------------------------------------------------

# 📅 Recommended Monthly Routine

1.  Beginning of month → Update income if needed
2.  During month → Log expenses
3.  End of month → Add savings
4.  Review financial status
5.  Adjust categories if needed

------------------------------------------------------------------------

# 🚨 Warning Signs

You should adjust your finances if:

-   Expenses exceed income
-   Savings rate below 10%
-   No emergency fund
-   Rent above 40% of income
-   Balance becomes negative

------------------------------------------------------------------------

# 📁 Data & Privacy

-   Each user has a separate JSON file
-   Data stored locally (not shared)
-   Restarting the bot does NOT delete data
-   Multiple users are isolated from each other

------------------------------------------------------------------------

# ❓ Frequently Asked Questions

**Q: Should I include housing allowance in income?**\
Yes --- include total take-home pay.

**Q: My expenses change monthly. What should I do?**\
Use your average monthly spending.

**Q: Can I reset everything?**\
Delete your JSON file from the user_profiles folder.

**Q: Is this a replacement for financial advisors?**\
No. This is a budgeting tracker tool.

**Q: Can families use it?**\
Yes --- combine total household income and expenses.

# 🛠 Troubleshooting

If something isn't working:

-   Use `/start` to reload
-   Ensure inputs are numbers only
-   Restart the bot if needed
-   Check internet connection
